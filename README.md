# 📱 Calculating Machine (안드로이드 계산기)

안드로이드 스튜디오로 제작한 간단하고 직관적인 계산기 프로젝트입니다.

## 🛠 기술 스택
- **Language**: Kotlin
- **IDE**: Android Studio
- **Build Tool**: Gradle (Kotlin DSL)
- **Min SDK**: 24 (Android 7.0)

## 📂 설치 및 실행 방법
1. 이 저장소를 클론(Clone)합니다.

   ```bash
   git clone https://github.com/GrayOM/android_calculator_basic.git
   ```
3. Android Studio에서 프로젝트를 불러옵니다.
4. 기기 또는 에뮬레이터에서 Run을 클릭하여 실행합니다.
